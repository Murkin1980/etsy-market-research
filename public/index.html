Защищённое соединение

Подключить панель

В Secret Manager откройте etsy-production-env и вставьте только значение строки API_KEY. Ключ хранится только в текущей вкладке браузера.

Ключ не записывается в cookies, URL или журналы сервера.

Источник данных

Подключить Etsy Open API

Введите данные приложения Etsy. Перед сохранением сервер проверит их через официальный API.

Ключ хранится на сервере в зашифрованном виде и никогда не возвращается в браузер.
